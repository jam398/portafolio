## Tech Stack

This portfolio is built with:

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- ESLint
- Playwright for browser QA checks

## Local Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the production version:

```bash
npm run build
```

Start the production build:

```bash
npm run start
```

Run lint:

```bash
npm run lint
```
